# 在线协作文档智能处理与校验系统

基于 Claude Code 多 Agent 协作的智能文档处理平台。

## 核心功能

- **文档解析 Agent**：清洗与脱敏原始文档（Markdown、PDF、录音转写）
- **结构校验 Agent**：基于企业文档规范进行长链推理校验
- **演示汇报生成 Agent**：一键转换为交互式 HTML

## 技术架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 文档解析    │ -> │ 结构校验    │ -> │ 演示生成    │
│   Agent     │    │   Agent     │    │   Agent     │
└─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │
       └──────────────────┴──────────────────┘
                      │
              ┌───────────────┐
              │  协作引擎     │
              │ (编排器)      │
              └───────────────┘
```
